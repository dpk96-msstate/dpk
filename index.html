Research Program & Scientific Contributions
Research focuses on software maintenance, software evolution, refactoring, unit testing, and empirical software engineering. Dissertation research investigates the characteristics that enable unit test suites to remain effective during production code refactoring. Contributions include the development and evaluation of a conceptual framework for test-suite resilience, systematic analysis of maintainability-related development practices, and practical guidance for improving software maintainability and evolution.
